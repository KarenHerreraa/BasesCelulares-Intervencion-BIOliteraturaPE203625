# BasesCelulares-Intervencion-BIOliteraturaPE203625
Este repositorio reune toda la información sobre las intervenciones de Bases Celulares del proyecto BIOliteraturaPE203625, a cargo del Dr. Israel Muñoz Velasco 

La página general de las intervenciones es: https://karenherreraa.github.io/Intervencion-BIOliteraturaPE203625/

La intervención se implementó en la asignatura Bases Celulares con estudiantes de licenciatura en Biología. Se diseñó una actividad diagnóstica y formativa, centrada en el uso de literatura científica y el manejo de referencias bibliográficas.
